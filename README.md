## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/techstore.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm run server
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server folder.

```
PORT=5000
MONGO_URI=Your MongoDB URI
JWT_SECRET=Your Secret Key
```
